# Глоссарий – Лабораторная работа 02

## Новые термины (добавлены при выполнении ЛР02)

1. **Глобальная интерпретация (Global interpretability)** – объяснение поведения модели в целом, например, какие признаки в среднем наиболее важны. Используется для доверия к модели до внедрения.

2. **Native importance** – важность признаков, вычисляемая внутренними механизмами модели (коэффициенты линейной модели, важности из RandomForest). Быстрая, но может быть смещённой.

3. **Permutation importance** – метод, измеряющий падение качества модели после случайного перемешивания значений признака. Не зависит от типа модели и даёт робастную оценку.

4. **Partial dependence plot (PDP)** – график, показывающий зависимость среднего предсказания от значения одного признака, усреднённого по всем остальным. Помогает увидеть нелинейные эффекты.

5. **Perturbation analysis** – локальный метод объяснения: небольшие изменения признаков и наблюдение за изменением предсказания. Показывает, какие признаки «чувствительны» для конкретного объекта.

6. **Counterfactual explanation** – объяснение вида «если бы значение признака X было другим, предсказание изменилось бы». Близко к perturbation, но ищет минимальное изменение для смены класса.

7. **Сегментный анализ ошибок (error segmentation)** – разбиение выборки на сегменты (по возрасту, доходу) и вычисление ошибок внутри каждого сегмента для выявления систематических смещений.

_Все термины использованы при выполнении ноутбуков и отражены в отчёте._
