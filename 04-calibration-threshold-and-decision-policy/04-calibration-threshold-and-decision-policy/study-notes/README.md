# Учебные заметки (обязательная часть самостоятельной работы)

Эта папка нужна для заметок по калибровке, порогам и стоимости ошибок.

## Минимальные требования
- `glossary.md` пополняется по ходу обоих ноутбуков.
- На каждый ноутбук: минимум 3 новых термина.
- Для ключевых понятий добавляйте:
  - простое объяснение;
  - одну формулу (где уместно);
  - пример интерпретации на данных ЛР;
  - источник.

## Когда выносить в отдельную заметку
- Нужно разделить ранжирование (discrimination) и калибровку (calibration).
- Нужно подробно разобрать формулы Brier/LogLoss/ECE.
- Нужно объяснить компромисс между порогом, стоимостью и полнотой.
- Нужно обосновать ограничение-страховку (guardrail) для правила решения.

Совет по стилю:
- сначала объясните идею простыми словами;
- затем дайте формулу;
- после формулы обязательно покажите маленький пример интерпретации.
